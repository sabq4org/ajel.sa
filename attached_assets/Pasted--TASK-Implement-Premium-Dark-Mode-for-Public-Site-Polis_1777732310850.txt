═══════════════════════════════════════════════
1️⃣ DARK MODE COLOR SYSTEM
═══════════════════════════════════════════════

❌ WHAT TO AVOID:
- Pure black backgrounds (#000) — harsh on eyes, hides depth
- Pure white text on black — too much contrast, causes halation
- Same colors as light mode just darker — kills mood
- Bright saturated colors that "bleed" on dark
- Removing all shadows (depth disappears)

✅ DO THIS INSTEAD:

LAYERED BACKGROUND SYSTEM (creates depth):
- Layer 0 (deepest): #0a0a0d (almost black, slight blue tint)
- Layer 1 (page bg): #0f0f12 (slate-950 vibe)
- Layer 2 (cards):   #18181b (zinc-900)
- Layer 3 (elevated): #27272a (zinc-800)
- Layer 4 (hover):    #3f3f46 (zinc-700)

BORDERS:
- Default: rgba(255,255,255,0.06) — barely visible
- Strong:  rgba(255,255,255,0.10)
- Brand:   {brand}-500/30

TEXT:
- Primary:   #FAFAFA (off-white, never pure white)
- Secondary: #D4D4D8 (zinc-300)
- Muted:     #A1A1AA (zinc-400)
- Faint:     #71717A (zinc-500)

BRAND COLORS in DARK:
- Burgundy adjusted: lifted saturation +10%, lifted lightness +15% for visibility
  - Use #C9384A or rose-500 for active states (instead of dark burgundy)
- Accents (gold/amber): keep slightly desaturated to avoid "bleeding"

SEMANTIC COLORS in DARK:
- Success: emerald-400 (lifted from emerald-500)
- Warning: amber-400
- Danger:  rose-400
- Info:    sky-400

4️⃣ TRANSITION & PERSISTENCE
═══════════════════════════════════════════════

GLOBAL TRANSITIONS (in CSS or globals.css):
```css
* {
  transition: background-color 200ms ease, 
              border-color 200ms ease, 
              color 200ms ease;
}⚠️ But exclude transforms and other properties to prevent jank!

PREVENT FLASH (in <head> of layout):

<script dangerouslySetInnerHTML={{
  __html: `
    try {
      const theme = localStorage.getItem('theme') || 'system';
      const resolved = theme === 'system' 
        ? (window.matchMedia('(prefers-color-scheme: dark)').matches ? 'dark' : 'light')
        : theme;
      document.documentElement.classList.toggle('dark', resolved === 'dark');
      document.documentElement.style.colorScheme = resolved;
    } catch {}
  `
}} />PERSIST PREFERENCE:

• Save to localStorage on toggle
• Sync between admin and public site (same key: theme)
• Provide a setting in admin to set default theme for the entire site

5️⃣ ACCESSIBILITY
═══════════════════════════════════════════════

REQUIREMENTS:

• Contrast ratios meet WCAG AA (4.5:1 for body text, 3:1 for large text)
• Focus rings visible: ring-2 ring-{brand}-500 ring-offset-2 ring-offset-zinc-950
• Don't rely solely on color (e.g. error states have icons too)
• Respect prefers-reduced-motion — disable transitions for users who set this
